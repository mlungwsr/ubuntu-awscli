# ubuntu-awscli
Dockerfile with Ubuntu image and awscli

### Packages
Install awscli on ubuntu 16.04

### Run
git clone https://github.com/mlungwsr/ubuntu-awscli.git

cd ubuntu-awscli/

docker build -t ubuntu-awscli .

docker run -dit ubuntu-awscli
